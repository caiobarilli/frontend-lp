# Frontend LP

| app    | host | port |
| -------------- | ---- | ---- |
| **nextjs**    |localhost | 3000 |

<br/>

### Pré-requisitos
#
Para começar, certifique-se de ter o [NodeJS](https://nodejs.org/en/) no seu sistema.

<br />

### Download
#
Faça o download do projeto com o comando:

```sh
git clone git@github.com:caiobarilli/frontend-lp.git
```

### Instalação
#
```sh
yarn
```

### Nextjs
#
```sh
yarn run dev
```
